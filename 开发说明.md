## 需要手动完成的步骤

### 1. 环境准备

**需要安装的软件：**
- Node.js 18+
- Java 17+
- Maven 3.8+
- MySQL 8.0+

### 2. 数据库设置

```bash
# 登录MySQL
mysql -u root -p

# 创建数据库和表
source database/init.sql
```

### 3. 后端启动

```bash
cd fund-tracker-backend
mvn clean compile
mvn spring-boot:run
```

### 4. 前端启动

```bash
cd fund-tracker-frontend
npm install
npm run dev
```

## 代码结构说明

### 前端主要文件

- `src/pages/Dashboard.vue` - 仪表板页面
- `src/pages/FundList.vue` - 基金列表页面
- `src/pages/MyHoldings.vue` - 持仓管理页面

- `src/components/ProfitChart.vue` - 收益图表组件
- `src/api/fund.ts` - API接口定义

### 后端主要文件

- `controller/FundController.java` - API控制器
- `service/FundService.java` - 基金服务
- `service/ProfitService.java` - 收益计算服务
- `repository/FundRepository.java` - 基金数据访问
- `repository/FundHoldingRepository.java` - 持仓数据访问
- `entity/Fund.java` - 基金实体
- `entity/FundHolding.java` - 持仓实体

## 功能演示

### 添加基金持仓
1. 访问 http://localhost:3000/funds
2. 在基金列表页面点击"添加持仓"按钮
3. 填写基金代码、买入价格、份额和日期
4. 提交后可在"我的持仓"页面查看

### 查看收益
1. 访问 http://localhost:3000/ 查看仪表板
2. 查看总投资、当前市值、总收益和收益率
3. 在"我的持仓"页面查看详细持仓信息

### 删除持仓
1. 访问 http://localhost:3000/holdings
2. 在持仓列表中找到要删除的记录
3. 点击"删除"按钮确认删除

## 技术实现要点

### 1. 模拟数据处理
- 基金数据使用模拟数据生成
- 价格更新使用定时任务模拟
- 用户系统暂未实现，使用固定用户ID

### 2. 收益计算
- 总收益 = 当前市值 - 总投资成本
- 收益率 = 总收益 / 总投资成本 * 100%
- 实时计算每个持仓的收益情况

### 3. 响应式设计
- 使用Element Plus组件库
- 支持不同屏幕尺寸
- 移动端友好的界面设计

## 常见问题

### Q: 如何添加新的基金？
A: 在添加持仓时输入基金代码，系统会自动创建基金记录（使用模拟数据）

### Q: 价格数据如何更新？
A: 后端每5分钟自动更新一次价格（模拟），实际项目中需要接入真实的基金数据API

### Q: 为什么看不到收益变化？
A: 一期版本使用模拟数据，收益变化也是模拟的。实际项目中会接入实时基金价格
